1. Kaj je OS?
To je temeljna programska oprema, ki deluje kot vmesnik med uporabnikom in strojno opremo računalnika, omogoča zagon aplikacij ter upravlja vire, kot so procesor, pomnilnik (RAM) in trdi disk. Brez njega računalnik ne more delovati. Najbolj znani primeri so Windows, Linux, macOS, Android in iOS.

2. Tri naloge?
Časovno razporejanje opravil,
Reševanje konfliktnih situacij (sočasni dostopi do virov),
Optimizacija in nadzira uporabe virov.
